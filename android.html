Tasks

Output in a loop

Print the column of numbers 16 through 23, skipping the number 21. Use a for loop and do the same with a while loop.

Code for the for loop:

for ($i = 16; $i <= 23; $i++) {
if ($i == 21)
continue;
echo $i;
}

Code for the for loop:

$i = 16;
while ($i <= 23) {
if ($i == 21) {
$i++;
continue;
}
echo $i;
$i++;
}
